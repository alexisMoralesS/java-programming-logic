> 💡 **Colección de ejercicios de lógica de programación en Java** diseñados para ayudar a desarrolladores a mejorar sus habilidades de resolución de problemas y pensamiento algorítmico.

## 📋 Descripción

Este repositorio contiene una serie de ejercicios prácticos de lógica de programación implementados en Java. Cada ejercicio está diseñado para fortalecer diferentes aspectos del pensamiento computacional, desde operaciones básicas hasta algoritmos más complejos.

### 🎯 Objetivo

Proporcionar un recurso educativo para:
- Desarrolladores principiantes que desean mejorar su lógica de programación
- Estudiantes que preparan entrevistas técnicas
- Programadores que buscan practicar y reforzar conceptos fundamentales
- Autodidactas en busca de ejercicios estructurados

## 🚀 Comenzando

### Prerrequisitos

Antes de comenzar, asegúrate de tener instalado:

- **Apache NetBeans** 12.0 o superior ([Descargar](https://netbeans.apache.org/download/))
- **Java JDK 17** o superior (NetBeans puede ayudarte a instalarlo)
- **Maven** ya viene integrado con NetBeans
- **Git** (opcional, para clonar el repositorio)

### 📥 Instalación y Configuración

#### Opción 1: Usar NetBeans (Recomendado)

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/alexisMoralesS/java-programming-logic.git
   ```

2. **Abre el proyecto en NetBeans:**
   - Abre NetBeans
   - Ve a `File > Open Project` (o presiona `Ctrl+Shift+O`)
   - Navega hasta la carpeta `java-programming-logic`
   - Selecciona el proyecto y haz clic en `Open Project`
   - NetBeans detectará automáticamente que es un proyecto Maven

3. **NetBeans compilará automáticamente el proyecto**
   - Si no lo hace, haz clic derecho en el proyecto → `Build`

### ▶️ Ejecutar los Ejercicios

#### En NetBeans:

1. **Navega hasta el ejercicio:**
   - Expande `Source Packages` en el panel de proyectos
   - Navega a `com.ejercicio1` → `Ejecutora.java`

2. **Ejecuta el archivo:**
   - Haz clic derecho en `Ejecutora.java`
   - Selecciona `Run File` (o presiona `Shift+F6`)
   - La salida aparecerá en la ventana de `Output` de NetBeans

3. **Interactúa con el programa:**
   - Ingresa los datos que el programa solicite en la consola de Output

### 🔧 Características de NetBeans para este Proyecto

- **Auto-completado inteligente**: NetBeans te sugerirá métodos y variables mientras escribes
- **Detección de errores en tiempo real**: Los errores se marcan inmediatamente
- **Depuración visual**: Usa breakpoints para inspeccionar el código paso a paso
- **Refactorización**: Renombra variables y métodos de forma segura en todo el proyecto
- **Integración con Git**: Realiza commits y push directamente desde NetBeans

## 🤝 Contribuir

¡Las contribuciones son bienvenidas! Si deseas agregar nuevos ejercicios o mejorar los existentes:

1. **Fork** el proyecto
2. Crea una **rama** para tu ejercicio (`git checkout -b ejercicio/nuevo-ejercicio`)
3. **Documenta** tu ejercicio con comentarios claros
4. Realiza **commit** de tus cambios (`git commit -m 'Add: nuevo ejercicio sobre ordenamiento'`)
5. **Push** a la rama (`git push origin ejercicio/nuevo-ejercicio`)
6. Abre un **Pull Request**

### 📐 Guía de Estilo

Al contribuir, por favor sigue estas convenciones:
- Usa nombres de paquetes descriptivos: `com.ejercicioX`
- Incluye comentarios explicativos al inicio de cada clase
- Especifica restricciones, ejemplos y conceptos practicados
- Mantén el código limpio y legible (usa `Alt + Shift + F` en NetBeans para formatear)
- Agrega validación de entrada cuando sea necesario
- Asegúrate de que el proyecto compile sin errores antes de hacer commit
